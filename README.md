# Проект доски вакансий

## Описание

Реализация демонстрационного REST API для доски вакансий с валидацией

### Технологии

- Nest.JS v10
- TypeORM
- Typescript
- Postgres

## Установка и запуск приложения

1. Клонировать репозиторий

   ```shell
   git clone
   ```

2. Запустить команду

   ```shell
   docker-compose up
   ```
